Integrates the WooCommerce Coupons system with the Affiliates Manager plugin

== Description ==

This addon allows you to track affiliate commission via WooCommerce discount codes or coupons.

The Affiliates Manager is a WordPress plugin that allows you to create a complete affiliate system within your WordPress website (without requiring any third party services).

If you are using WooCommerce plugin and you want to integrate it's discount coupons with the [Affiliates Manager Plugin](https://wordpress.org/plugins/affiliates-manager/) then use this addon plugin.

You can find detailed usage instructions for this addon [here](https://wpaffiliatemanager.com/tracking-affiliate-commission-using-woocommerce-coupons-or-discount-codes/)

== Installation ==

1. Upload the "affiliatemgr-woocommerce-coupons-integration.zip" file via the WordPress's plugin uploader interface (Plugins -> Add New -> Upload)
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to the Woo Coupons interface from the Affiliates Manager plugin's admin dashboard menu.

Configure different affiliate commission levels for your WooCommerce products

== Description ==

This addon allows you to specify product specific commissions for your WooCommerce products.

[Affiliates Manager](https://wordpress.org/plugins/affiliates-manager/) is a WordPress plugin that allows you to create a complete affiliate system within your WordPress website (without requiring any third party services).

If you are using WooCommerce plugin and you want to specify commission rate on a per product basis then use this addon. 

You can find detailed usage instructions for this addon [here](https://wpaffiliatemanager.com/product-specific-affiliate-commission-for-woocommerce-products/)

== Installation ==

1. Upload the "affiliatemgr-woo-product-specific-commission.zip" file via the WordPress's plugin uploader interface (Plugins -> Add New -> Upload)
2. Activate the addon through the "Plugins" menu in WordPress.
3. Go to the WooCommerce Edit Product menu to specify commission for a particular product.
